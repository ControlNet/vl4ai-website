The archived page defined this project around a higher-level navigation pipeline in which learned scene representations support both local navigation and global planning in previously unseen environments.

## Legacy project aims

1. Develop scene-layout representations and deep-learning methods that capture high-level structure useful for local navigation and global planning.
2. Link higher-level scene representations to planning, actively explore an environment, and update imagined and real scene representations online.
3. Integrate the first two aims into a navigation method based on high-level visual entities in previously unmapped environments.

## Legacy skills summary

The old page requested strength in machine learning, computer vision, and robotics; solid applied mathematics or probability foundations; programming in Python and C/C++; and, desirably, ROS plus prior experience in 3D vision, SLAM, navigation, or reconstruction.
